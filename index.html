TGIS Collector User Guide

Welcome to the TGIS Collector User Guide! This comprehensive guide will help you understand and effectively use the TGIS Collector mobile application, designed specifically for GIS data collection.

Getting Started

Initial Setup

  1. Launch the TGIS Collector app
  2. Grant necessary permissions (location, camera, storage)
  3. Select your preferred theme (light/dark mode)

Key Features

Geometry Selection

The app supports multiple geometry types for data collection:

  • Point: Capture specific locations
  • Line: (Coming soon) Record linear features like roads or paths
  • Polygon: (Coming soon) Map area-based features like lakes or property boundaries

Quick Capture

  • Simple one-tap location capture
  • Real-time accuracy display in meters
  • Automatic point naming (Point 1, Point 2, etc.)

Flexible Capture

  • Detailed data entry interface
  • Customizable input fields
  • Real-time coordinates and accuracy display in meters
  • Photo attachment capability

Data Collection

Adding Points

Quick Capture

  1. Tap the "Capture" button
  2. Verify location accuracy
  3. Point is automatically named and added to point list.
  4. Tap "Save" button to save your points.

Detailed Capture

  1. Tap "Add New Point"
  2. Enter required information
  3. Attach photos if needed
  4. Tap "Add Point" to add the point to list
  5. Tap "Save" button to save your points.

Custom Input Questions

Access the settings menu from the sidebar, to add custom fields for the Detailed Capture:

  • Text input
  • Numeric input
  • Email fields
  • Radio buttons
  • Dropdown menus

    • You can delete any added custom field from the list of added custom groups on the same popup window.

Data Management

Point List

  • View all captured points
  • Click on any point to view more information
  • Edit point details
  • Delete individual points
  • Clear all points

Save Options

  • Save as CSV
  • Save as GeoJSON
  • Access saved information in "Data" section or in the sidebar via "My Data" option

File Management

  • View saved files
  • Click image to view
  • Select and share files via multiple platforms using the social sharing capability of android:
  • Delete files individually or in bulk

Privacy & Security

Data Storage

  • All data stored locally on device
  • No cloud storage or third-party transmission
  • Complete control over data sharing

Offline Functionality

  • Works without internet connection
  • Requires only location permissions
  • Full feature access offline

Customization

User Interface

  • Toggle between light and dark themes
  • Customizable input fields
  • Flexible data collection options

Troubleshooting

Common Issues

App not installed

  • Verify sufficient storage space.
  • Verify installation permissions.

Error getting location

  • Ensure GPS is enabled on youy device.
  • Check location permissions.
  • Turn off and on your Location.
  • Wait for your device to get your location.The application allows 30 seconds of acquiring location before displaying a timeout error.
  • Close the application, turn on Location then open the application again.
  • NB: Some mobile devices may have basic GPS support but may not function effectively in offline mode.

Poor location Accuracy

  • Wait for better GPS signal
  • Check if you are near obstacles, e.g buildings, trees, etc.

Attaching photos

  • Ensure proper permissions

Data Export

  • Verify sufficient storage space
  • Check file format compatibility
  • Ensure proper permissions
